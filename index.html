Countix

Countix is a simple counter app for iPhone, iPad, and Apple Watch. It is designed for quick quantity counting, saving results, continuing counts across devices, and exporting saved data when needed.

Features

  • Start a new count quickly
  • Name a count
  • Continue an active or saved count
  • Save counts for later
  • View, edit, reorder, and delete saved counts
  • Export saved counts as PDF or CSV
  • Share exported files
  • Choose the app language
  • Choose the app date format, including system default and common international formats
  • Enable optional iCloud Sync for saved counts and selected settings between iPhone and iPad
  • Choose app appearance: system, light, or dark
  • Enable or disable sounds and haptics
  • Use Countix on iPhone, iPad, and Apple Watch
  • Sync active count state, saved counts, language, date format, sound, and haptic settings between iPhone and Apple Watch

Languages

Countix supports English, Finnish, German, Spanish, French, Italian, Dutch, Portuguese (Brazil), Swedish, Japanese, Korean, and Simplified Chinese. If the device language is not supported, Countix uses English.

Date Formats

Saved counts and exports can use the system date format or a selected format such as 31.12.2026, 31/12/2026, 12/31/2026, 2026/12/31, or 2026-12-31.

Exports

PDF and CSV exports use the selected app language and date format. CSV exports preserve saved count names and values in a spreadsheet-friendly format.

iCloud Sync

iCloud Sync is optional. When enabled, Countix can sync saved counts, deletion history, the active count state, and selected app settings between iPhone and iPad through Apple’s iCloud key-value storage.

Appearance

The appearance setting changes only Countix. It does not change the system appearance of the iPhone, iPad, or Apple Watch.

Apple Watch

The Apple Watch app can start, continue, save, edit, and delete counts. Selected settings and saved count data sync with the paired iPhone when available.
